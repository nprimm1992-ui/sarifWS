## P0 — User-visible / security / build-breaking

### P0-01 · Lobby Three.js scene builds on every route (Perf)
- **Where:** `initLobby()` in [src/scripts/lobby-scene.js:1412](src/scripts/lobby-scene.js); scheduled from [src/layouts/Base.astro:168](src/layouts/Base.astro) on every `astro:page-load`. `_isLanding` flag only gates logo/bloom, not the WebGL scene build at [src/scripts/lobby-scene.js:1036-1147](src/scripts/lobby-scene.js).
- **Impact:** Inner pages (About, Services, etc.) pay Three.js + composer + rAF cost while About also runs the UCIM iframe's WebGL. Highest TBT / battery regression on mobile.
- **Fix:** Early-return from `initLobby()` when `!isLandingPath()`; ensure `cleanup()` runs on `astro:before-swap`.

### P0-02 · Audio listeners stack across view transitions (Perf + code quality)
- **Where:** `initAudio()` at [src/scripts/audio.js:164-186](src/scripts/audio.js) adds capturing `document` `mouseenter` + `click` listeners every call; [src/layouts/Base.astro:189-201](src/layouts/Base.astro) invokes on `DOMContentLoaded` **and** `astro:page-load` with no teardown. `atmosphere.js` does have cleanup; audio does not.
- **Impact:** Duplicate tones and extra main-thread work after any in-app navigation.
- **Fix:** Module-level guard flag + `AbortController` teardown on `astro:before-swap`, mirroring the atmosphere pattern.

### P0-03 · UCIM iframe is `loading="eager"` (Perf)
- **Where:** [src/pages/about.astro:105-112](src/pages/about.astro). The visualizer under `public/ucim-visualizer/` is a full React + WebGL app.
- **Impact:** UCIM boot competes with the rest of About (and the lobby scene from P0-01), producing a perceptible stall.
- **Fix:** `loading="lazy"` + IntersectionObserver-gated `src` swap. Keep `transition:persist="about-ucim-embed"` unchanged.

### P0-04 · `/api/contact` has no rate limit and `Access-Control-Allow-Origin: *` (Security)
- **Where:** [functions/api/contact.js:20-26](functions/api/contact.js) returns `*`; the hot path at [functions/api/contact.js:100-109](functions/api/contact.js) calls `GOOGLE_SCRIPT_URL` with no D1 counter. Meanwhile [functions/api/transmit.js:88-124](functions/api/transmit.js) caps 10 per rolling day.
- **Impact:** Mail-relay abuse bypass. Anyone can loop `/api/contact` from any origin.
- **Fix:** Reuse `hashIp`, `corsPreflight()`, `jsonHeaders()` from [functions/api/_shared/validate.js:179-188](functions/api/_shared/validate.js). Reflect `Origin` against a `https://sarifconsulting.ai` allowlist. Use same D1 counter with an `intent` marker.

### P0-05 · OG + favicon PNG assets missing from `public/` (SEO)
- **Where:** [src/layouts/Base.astro:23-24, 51-59](src/layouts/Base.astro) references `/og-image.png`, `/phoenix-emblem.png`, `/apple-touch-icon.png`. None exist in `public/` or `dist/`. Generator scripts: [scripts/generate-og-image.mjs:9](scripts/generate-og-image.mjs), [scripts/optimize-favicon.mjs:12](scripts/optimize-favicon.mjs).
- **Impact:** Broken social-preview cards; broken favicon tabs / PWA icon.
- **Fix:** Generate and commit the three PNGs to `public/`; shrink the `rel="icon"` link from 1024×1024 to 32/48 favicons; keep 1024 only for `apple-touch-icon`/PWA.

### P0-06 · Praxis subscribe form silent on empty/invalid submit (UX)
- **Where:** [src/pages/praxis.astro:471-475](src/pages/praxis.astro). Early return on `!email`, no `reportValidity()`, no error region.
- **Impact:** Keyboard/mobile users get a dead button. Dead-end.
- **Fix:** Invoke `emailInput.reportValidity()` before early-return; write the error region (`#subscribe-error-summary`) for server-side 400s.

### P0-07 · Contact receipt status over-claims vs body (UX)
- **Where:** [src/pages/contact.astro:148-151, 164-177](src/pages/contact.astro). Status header reads "Transmission acknowledged" while body promises a reply later.
- **Impact:** After the SLA-copy sweep, "acknowledged" reads as "we already replied."
- **Fix:** Change header to "Transmission received" and keep the "reply will arrive" bubble. No new state.

## P1 — Significant hardening / resilience

### Security

1. **IP-hash salt falls back to a known string.** [functions/api/_shared/validate.js:126-137](functions/api/_shared/validate.js) falls back to `'sarif-consulting-ip-hash-v1'`. → Require `IP_HASH_BASE_SALT` in production; throw at first use if missing.
2. **Outbound `GOOGLE_SCRIPT_URL` fetch has no host allowlist.** [functions/api/transmit.js:211-221](functions/api/transmit.js), [functions/api/contact.js:100-109](functions/api/contact.js). → Parse URL, assert host ∈ allowlist.
3. **`JENSEN_PICKUP_TOKEN` grants read + claim + draft in one.** [functions/api/_shared/pickup-auth.js](functions/api/_shared/pickup-auth.js). → Split read vs write scopes.
4. **Draft endpoint silently overwrites.** [functions/api/pickup/[id]/draft.js:51-52](functions/api/pickup/[id]/draft.js). → Reject if `status ∈ {drafted, sent}` unless explicit `force` with monotonic trace ID.
5. **No per-email rate limit.** → Secondary D1 counter keyed on email hash with its own salt.
6. **No Content-Security-Policy.** [public/_headers](public/_headers) has `X-Frame-Options` / `X-Content-Type-Options` / `Referrer-Policy` / `Permissions-Policy` but no CSP. → Add strict CSP (fonts + Cloudflare Insights + `'self'`).
7. **UCIM iframe has no `sandbox` attribute.** [src/pages/about.astro:105-112](src/pages/about.astro). → `sandbox="allow-scripts allow-same-origin"`.
8. **No HSTS in _headers.** → Add `Strict-Transport-Security: max-age=31536000; includeSubDomains; preload`.

### Performance

9. **Google Fonts overweight.** [src/layouts/Base.astro:63-75](src/layouts/Base.astro) loads Orbitron 400–900 + Inter + Space Grotesk. → Trim to weights actually used.
10. **`chunkSizeWarningLimit: 800` hides regressions.** [astro.config.mjs:15](astro.config.mjs). → Drop to 500; explicit `manualChunks` if tripped.
11. **Hero `.hero__scroll-line` runs infinite.** [src/pages/index.astro](src/pages/index.astro). → IntersectionObserver `animation-play-state` toggle.
12. **Mixed viewport units.** [src/pages/about.astro:205-218](src/pages/about.astro) `62vh`; [src/pages/404.astro:32-36](src/pages/404.astro) `80vh`. → Use `svh` / `dvh` consistently.

### SEO

13. **Trailing-slash mismatch.** Sitemap emits `/about/`, Nav links to `/about`. → `trailingSlash: 'always'` + internal link sweep.
14. **404 is indexable.** [src/pages/404.astro:5-8](src/pages/404.astro). → Pass `noindex` via Base prop.
15. **Static JSON-LD on every page.** [src/layouts/Base.astro:87-103](src/layouts/Base.astro). → Optional `jsonLd` prop for page-level `WebPage`/`Article`; populate `sameAs`.
16. **Engagements meta description very short.** [src/pages/engagements.astro:128-130](src/pages/engagements.astro). → Expand to one real paragraph.
17. **`robots.txt` allows `/api/` probing.** [public/robots.txt](public/robots.txt). → `Disallow: /api/`.

### UX

18. **Contact error receipt has no "Try again."** [src/pages/contact.astro:187-200](src/pages/contact.astro). → Primary retry CTA; restores focus.
19. **Subscribe 429 handling is absent.** [src/pages/praxis.astro:425-444](src/pages/praxis.astro) vs contact's matrix. → Shared `form-errors.js`.
20. **Mobile nav focuses first route link, not close.** [src/scripts/nav.js:48-50](src/scripts/nav.js). → Focus close button first for `aria-modal` pattern.
21. **Cookie dismiss does not restore focus source.** [src/components/Footer.astro:99-101](src/components/Footer.astro). → Capture `activeElement` on show, restore on dismiss.
22. **Honeypot pass renders full success UI.** [src/pages/contact.astro:1546-1552](src/pages/contact.astro). → Soft-fail, generic no-op.
23. **404 recovery nav omits About/Praxis.** [src/pages/404.astro:20-26](src/pages/404.astro).
